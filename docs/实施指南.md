# 实施指南（Coding Agent 执行清单）

> **目标受众**：Claude Code / Cursor / Copilot 等 AI coding agent
> **使用方式**：按阶段顺序逐任务执行，每个任务包含明确的改动点、验证命令、完成标准
> **原则**：外科手术式修改，只改必要部分，匹配现有代码风格，全程保持 `npm run check` 通过

---

## 如何使用本指南

1. **按阶段顺序执行**：阶段 0 → 阶段 1 → 阶段 2（后续按需）
2. **每个任务独立**：可单独 commit，可回退
3. **验证前置**：每个任务完成后立即运行验证命令，不通过不进入下一任务
4. **保持测试绿**：任何改动后运行 `npm run check`，红则立即修复
