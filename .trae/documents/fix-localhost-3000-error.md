# 修复 localhost:3000 无法打开的问题

## 问题分析

从日志中看到：
1. 开发服务器正在运行 (`npm run dev`)
2. 出现 `MODULE_NOT_FOUND` 错误
3. 错误与 `.next/server/pages/_document.js` 相关
4. 频繁重新编译表明缓存问题

## 原因

`.next` 目录中的缓存与新增代码不兼容，需要清理后重新启动。

## 修复步骤

1. **停止当前开发服务器**
   - 终止正在运行的 `npm run dev` 进程

2. **清理 .next 缓存目录**
   - 删除 `.next` 目录

3. **重新启动开发服务器**
   - 运行 `npm run dev`

## 预期结果

清理缓存后重新启动，服务器应该能正常运行，访问 `http://localhost:3000` 应该能正常显示页面。
