💡 优化建议
建议 1：增加搜索功能
VitePress 内置支持搜索（基于预渲染的索引）
对技术博客很实用，方便读者查找内容
配置简单，几乎无额外成本
建议 2：RSS 订阅支持
使用 vitepress-plugin-feed 生成 RSS
方便读者订阅你的博客更新
符合"长期可维护"的目标
建议 3：图片处理策略
建议统一将图片放在 /public/images/ 或文章同级目录
考虑是否需要图片压缩优化（可在 CI 中自动处理）
建议 4：本地开发体验优化
添加 .nvmrc 固定 Node 版本
添加 .editorconfig 统一编辑器配置
考虑添加简单的 Git Hook（如提交前检查）
建议 5：备份策略
虽然 Git 已有版本控制，但建议：
定期导出静态构建产物
或使用 GitHub 的备份功能
